# Trabalho Prático 1 - Extração de Conhecimento de Dados Biológicos (2025/2026)

Este repositório contém os guiões, dados e relatórios referentes ao Trabalho Prático 1 da unidade curricular de Extração de Conhecimento de Dados Biológicos.
O objetivo deste projeto é a análise de um conjunto de dados de expressão genética utilizando o R e pacotes do Bioconductor.

## Conjunto de Dados Selecionado
**Kidney Renal Clear Cell Carcinoma (TCGA, Firehose Legacy)**
* **Fonte:** cBioPortal
* **Tipo de Dados:** Expressão genética (RNA-Seq) e Metadados Clínicos[cite: 19].
* **Dimensão:** > 300 amostras (534 amostras de expressão).

## Organização do Repositório

* `data/`: Diretória destinada ao armazenamento dos dados (ficheiros clínicos e matrizes de expressão descarregados).
* `scripts/`: Contém os ficheiros de código R utilizados em cada fase da análise.
* `reports/`: Contém os relatórios finais gerados em formato HTML através do R Markdown, documentando os resultados e as discussões.
